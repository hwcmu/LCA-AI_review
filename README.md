# Integrating machine learning into life cycle assessment: Review and future outlook

Reproducibility repository for the published open-access article:

> Wang, H. (2025). Integrating machine learning into life cycle assessment: Review and future outlook. *PLOS Climate*, 4(10), e0000732. https://doi.org/10.1371/journal.pclm.0000732

## Quick Links

- Published article: https://journals.plos.org/climate/article?id=10.1371/journal.pclm.0000732
- DOI: https://doi.org/10.1371/journal.pclm.0000732
- Repository: https://github.com/hwcmu/LCA-AI_review

## Repository Scope

This repository contains the bibliometric materials used in the review article
to map machine-learning applications in life cycle assessment (LCA). The article
organizes opportunities across the LCA workflow, including goal and scope
definition, life cycle inventory, life cycle impact assessment, and
interpretation.

The files here support the reproducible literature-map component of the paper.
They are not a full computational LCA model or a replacement for the article
text.

## Files

- `Literature from WOS.txt`: Web of Science bibliographic export used for the map.
- `map file.txt`: VOSviewer map export with item coordinates and cluster IDs.
- `network file.txt`: VOSviewer network export with links and weights.
- `network visualization.png`: VOSviewer network visualization.
- `overlaay visualization.png`: VOSviewer overlay visualization.

## Literature Search Strategy

- Database: Web of Science Core Collection
- Time span: 2015-2025
- Search concept: LCA-related terms combined with machine-learning and
  AI-related terms, with additional terms covering data gaps, uncertainty,
  modeling, prediction, optimization, and decision support.

The full query text is preserved in the original exported materials and
publication workflow notes.
